loadPageChunk

  • loadPageChunk(pageID: string, chunkNo?: undefined | number, cursor?: Cursor): Promise<LoadPageChunkReturns>
    • Defined in index.ts:688

    Make a POST request to /api/v3/loadPageChunk for one chunk of a page.

    remarks

    The chunk contains only top level blocks.

    Parameters

    • pageID: string

      The ID (with dashes) of a page.

    • Optional chunkNo: undefined | number

      The chunk number. Starting from 0, add 1 for each chunk received. Specify 0 or neglect this parameter for the first chunk.

    • Optional cursor: Cursor

      The Cursor returned by Notion in the last chunk. Neglect this parameter for the first chunk.

    Returns Promise<LoadPageChunkReturns>

    An object containing response data and error.
